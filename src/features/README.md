# Feature modules

- `telegram/` — авторизация Telegram Mini App.
- `ton/` — подключение TON кошелька (Tonkeeper/Tonhub) и базовые wallet-операции.
- `referrals/` — генерация и разбор реферальных ссылок Telegram-бота.
- `token/` — ZooToken (Jetton) сервис и конвертер очков в токены.
- `nft/` — NFT mint/transfer/breed API.
- `marketplace/` — UI маркетплейса NFT.
- `profile/` — UI профиля игрока (аватар, кошелёк, стейкинг, рефералка).
- `friends/`, `clans/`, `gamification/` — reserved for social/game loops.
